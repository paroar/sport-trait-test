## Installation

```bash
$ npm install
```

## Running infrastructure

```bash
# start docker-compose
$ npm run infra:up

# stop docker-compose
$ npm run infra:stop
```

## Running migrations

```bash
$ npm run m:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
